Chapters 2 and 3 definitions

Chapter 2 Terms

<h1>
This is the top level header
<h2>
This is the second level header a sub header to <h1>
<h3>
This is the third level header a sub header to <h2>
<h4>
This is the fourth level header a sub header to <h3>
<h5>
This is the fifth level header a sub header to <h4>
<h6>
This is the sixth level header a sub header to <h5>
<p>
This is a new paragraph tag to separate different pararaphs
<b>
This is a bold text tag to make text bold
<i>
This is an italics text tag to make text italic
<sup>
This is a super script text tag to make superscript text such as an exponent 42
<sub>
This is a subscript text tag to allow you to make subscript such has a formula H20
<br />
This is a line break it moves text to the next line
<hr />
This is a horizontal rule it adds a horizontal line as a divider
<srong>
This is also bolds text
<em>
This is adds emphasis to text which comes out italic
<blockquote>
This makes a section or paragraph a quote
<q>
This is adds quotes around text
<abbr>
This will allow you to abbreviate text
<cite>
This allows you to cite a source
<dfn>
This allows you to define a word or phrase
<address>
This will allow you to provide the address of the author
<ins>
This shows content inserted into text
<del>
This allows you to show text that has been deleted
<s>
This allows a strike through

Chapter 3 Terms

<ol>
This is an ordered list that will number items
<li>
This is a list item to put data in an ordered or unordered list
<dl>
This is a definition list allows you to create a list of definitions
<dt>
This is the term to be defined in a definition list
<dd>
This is the description of the definition term in the definition list
